# 心情日记 (Diary App)

Android 移动端软件开发期末大作业

## 技术栈

- Kotlin + Jetpack Compose + Material 3
- Room 数据库
- DataStore 偏好存储
- ViewModel + StateFlow
- Navigation Compose
- Retrofit + OkHttp 网络请求
- Coil 图片加载
- Kotlin Coroutines

## 功能

- 📝 创建/编辑/删除日记
- 😊 心情选择（开心/难过/平静/兴奋/生气/放松）
- 🔍 搜索和心情筛选
- 🌐 每日一言（ZenQuotes API）
- 🏷️ 标签分类
- 🌓 浅色/深色主题

## 运行

Android Studio 打开项目 → Sync Gradle → Run
最低 Android 8.0 (API 26)
